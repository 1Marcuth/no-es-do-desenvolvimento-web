Sobre

Quando começamos a cursar?

Começamos o nossas aulas depois de vários adiamentos consecutivos, e finalmente, em 22/05/2023 as aulas iniciaram.

Nosso curso terá uma duração de 18 meses, porém esse número de meses parece não considerar as férias, então aparentemente ficaremos quase dois anos.

O que foi ensinado até agora?

Hardware

Sobre Hardware apredendemos coisas sobre dispositivos de entrada, dispositovos de saída e dispositivos mistos. Também fizemos aulas prática de plugar e desplugar cabos de um gabinete.

Houve também uma aula sobre solda, onde nós (alunos) aprendemos um pouco sobre como manusear o ferro de solda para aplicá-lo em manutenção de componentes de um circuito eletrônico.

Recentemente tivemos uma aula prática para desmontar um gabinete e retirada de componentes como placa-mãe, CPU, HD e etc.

Software

Sobre Software aprendemos coisas como modelos do desenvolvimento de software como Modelo em Cascata, Modelo em Espiral, Modelo Incremental, Modelo de Prototipação e etc.

Recentemente também aprendemos sobre Levantamento de Requisitos de um Software e como coletar dados para criar a lógica de negócio de um software.

Nota: Provavelmente eu acabei esquecendo de algo ou errando alguma coisa.
